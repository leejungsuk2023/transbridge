  실시간 의료 통역 서비스
========================================

[접속 URL]
https://transbridge.vercel.app


========================================
1. 준비물
========================================

- 안드로이드 기기 또는 PC 1대 (접수대 비치)
- 유선 이어폰 (핸즈프리) 1개
  → 직원이 한쪽, 환자가 한쪽 나눠 착용
- 인터넷 연결 (Wi-Fi 권장)


========================================
2. 로그인
========================================

1) 브라우저에서 https://transbridge.vercel.app 접속
2) 발급받은 이메일 / 비밀번호 입력
3) "로그인" 버튼 클릭


========================================
3. 이어폰 세팅
========================================

1) 핸즈프리 이어폰을 기기에 연결
2) 직원이 한쪽, 환자가 한쪽 나눠 착용
3) 기기는 접수대 위에 놓고, 마이크가 양쪽 모두
   잘 잡히도록 배치


========================================
4. 통역 시작
========================================

1) 대시보드에서 환자 언어 선택
   - 지원 언어 (11개):
     태국어, 베트남어, 영어, 인도네시아어,
     스페인어, 몽골어, 광동어, 북경어,
     일본어, 프랑스어, 독일어

2) "통역 시작" 버튼 클릭
3) 마이크 허용 팝업 → "허용" 클릭
4) 화면 하단에 "듣는 중..." 표시되면 준비 완료


========================================
5. 통역 진행
========================================

- 직원이 한국어로 말하면
  → 선택한 언어로 번역 음성이 이어폰으로 나옴
  → 화면에 번역된 문장이 표시됨

- 환자가 외국어로 말하면
  → 한국어로 번역 음성이 이어폰으로 나옴
  → 화면에 한국어 번역이 표시됨

- 양방향 자동 감지이므로 버튼 조작 불필요
  그냥 말하면 자동으로 통역됩니다


========================================
6. 통역 종료
========================================

1) 우측 상단 "종료" 버튼 클릭
2) "통역을 종료하시겠습니까?" → 확인
3) 대시보드로 자동 이동

* 브라우저를 그냥 닫아도 세션은 자동 종료됩니다


========================================
7. 대시보드 확인
========================================

- 이번 달 통역 건수
- 총 사용 시간
- 언어별 통역 건수
- 최근 통역 이력 (날짜, 언어, 시간, 상태)


========================================
8. 사용 팁
========================================

- 조용한 환경에서 사용하세요
- 짧고 명확하게 한 문장씩 말하면 인식률이 높습니다
- 상대방이 말할 때는 잠시 기다려주세요
- 의료 용어 (보톡스, 필러, 레이저 등)는
  자동으로 정확한 번역이 적용됩니다
- 인터넷이 끊기면 "인터넷 연결 끊김" 안내가 뜨고
  복구되면 자동 재연결됩니다


========================================
9. 문제 해결
========================================

[마이크가 안 잡힐 때]
→ 브라우저 주소창 왼쪽 자물쇠 아이콘 클릭
→ 마이크 권한을 "허용"으로 변경

[연결이 끊겼을 때]
→ 자동으로 재연결을 시도합니다 (최대 5회)
→ 5회 실패 시 페이지를 새로고침 해주세요

[번역이 안 될 때]
→ 마이크 가까이에서 또렷하게 말해보세요
→ 주변 소음이 심하면 조용한 곳으로 이동

[로그인이 안 될 때]
→ 이메일/비밀번호를 다시 확인해주세요
→ 문제 지속 시 관리자에게 문의
